# Аналізатор частин мови

Цей застосунок аналізує введений користувачем текст і замінює слова на відповідні їм частини мови (іменник, дієслово, прикметник тощо), використовуючи бібліотеку "spaCy" для української мови. Також зберігається пунктуація.

## Функціонал

- Замінює слова на відповідні їм частини мови.
- Відображає результат у вигляді тексту зі збереженою пунктуацією.
- Має графічний інтерфейс на основі "Gradio".
- Підтримує завантаження тестових прикладів з файлів.

## Вимоги

- Python 3.7+
- "spacy"
- "gradio"
- Модель "uk_core_news_sm" для української мови

## Використання

1. Введіть текст у поле вводу або виберіть один з прикладів.
2. Натисніть кнопку "Аналізувати".
3. Результат буде відображено у полі виводу.

## Приклад

- Вхід: "День буде хорошим."
- Вихід: "іменник дієслово прикметник ."
